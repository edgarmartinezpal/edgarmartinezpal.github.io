Sobre Mí

Soy Edgar Martínez Palacios, apasionado por el desarrollo de software y la creación de aplicaciones robustas tanto para web como para múltiples plataformas.

Actualmente estoy cursando el Ciclo Superior de Desarrollo de Aplicaciones Multiplataforma (DAM) en Florida Campus Alzira (septiembre 2024 – junio 2026), donde profundizo cada día en nuevas tecnologías y buenas prácticas de programación.

Mi stack principal es MERN (MongoDB, Express, React, Node.js), complementado con bases de datos SQL como MySQL y PostgreSQL. Me gusta escribir código limpio, escalable y bien documentado.
